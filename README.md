# Pragmatic Intelligence: Inflectra's AI Pivot

This repository contains a [Reveal.js](https://revealjs.com/) based web presentation detailing Inflectra's strategic and measured pivot to AI-assistive software delivery. 

## 📖 Overview

Instead of rushing to bolt on chatbots to capitalize on industry trends, Inflectra took a pragmatic approach. This presentation outlines that journey, emphasizing:
1. **Measured Innovation:** Building stable, deterministic AI tools rather than reactive "Beta" features.
2. **Integrity in Tooling:** Positioning AI as an *assistive* co-pilot, not a pilot. 
3. **The Maven Loop:** Sourcing feature requirements directly from QA leads and DevOps architects in the Inflectra community.

The deck specifically highlights AI integrations within:
* **Spira®:** Human-Centric ALM (Requirement Generation, Risk Analysis, Test Case Drafting).
* **Rapise®:** Breathtaking UI Automation (AI Object Recognition, Self-Healing Locators, Intent Translation).
